# geoserver-leaflet-map

Interactive Web Map with GeoServer and Leaflet

## Geodaten

Die Shapefiles sind **nicht im Repository** enthalten
da sie zu groß für GitHub sind.

### Daten herunterladen

1. https://download.geofabrik.de/europe/germany/ öffnen
2. Regierungsbezirk Karlsruhe herunterladen
3. ZIP entpacken
4. Alle Dateien in den Ordner `data/Karlsruhe/` kopieren

### Benötigte Dateien

| Datei                          | Beschreibung |
| ------------------------------ | ------------ |
| gis_osm_roads_free_1.shp       | Straßennetz  |
| gis_osm_water_a_free_1.shp     | Gewässer     |
| gis_osm_buildings_a_free_1.shp | Gebäude      |
| gis_osm_places_free_1.shp      | Orte         |

### In GeoServer laden

Siehe Abschnitt "GeoServer einrichten" weiter unten.
